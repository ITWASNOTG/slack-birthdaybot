# Contributing to Slack BirthdayBot

Thanks for your interest in contributing! This project is open source and welcomes improvements of all kinds.

## How to Contribute

1. **Fork** the repository
2. **Create a branch** for your change (`git checkout -b my-feature`)
3. **Make your changes** and test them
4. **Commit** with a clear message (`git commit -m "Add: description of change"`)
5. **Push** to your fork (`git push origin my-feature`)
6. **Open a Pull Request** describing what you changed and why

## Ideas for Contributions

- Bug fixes or edge-case handling
- New blurb or GIF suggestions
- Documentation improvements
- Translations of the README
- New features from the [Ideas to Build On](#ideas-to-build-on) section in the README

## Guidelines

- Keep it simple. This project is meant to be easy to understand and modify.
- Don't commit secrets (webhook URLs, API keys, etc.).
- Test your changes with a real Google Sheet + Slack webhook before submitting.

## Questions?

Open an issue and we'll figure it out together.
